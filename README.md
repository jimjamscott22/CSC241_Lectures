# CSC 241 Lectures

- This repository contains lecture notes and materials for CSC 241, a course on data structures and algorithms. 
- The lectures cover various topics including linked lists, stacks, queues, trees, graphs, sorting algorithms, and more. 
- Each lecture is organized into a separate file, and the materials include explanations, examples, and code snippets to help students understand the concepts. 
- The repository is intended to be a resource for students taking the course, as well as for anyone interested in learning about data structures and algorithms. 

**Feel free to explore the materials and use them for your studies!**
